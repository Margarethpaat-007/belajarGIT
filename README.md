## Daftar Perintah Git yang Digunakan

### Membuat dan Mengelola Repository
- `git init` → Menginisialisasi repositori Git lokal.
- `git clone <URL_REPOSITORY>` → Meng-clone repositori dari GitHub.

### Manajemen Branch
- `git checkout -b <nama-branch>` → Membuat branch baru dan berpindah ke branch tersebut.
- `git checkout main` → Berpindah ke branch utama.
- `git merge <nama-branch>` → Menggabungkan branch ke branch utama.

### Commit dan Push ke GitHub
- `git add <nama-file>` → Menambahkan file ke staging area.
- `git add .` → Menambahkan semua file yang telah dimodifikasi.
- `git commit -m "Pesan commit"` → Menyimpan perubahan ke dalam repository.
- `git push origin <nama-branch>` → Mengirim perubahan ke GitHub.

### Sinkronisasi dengan Remote Repository
- `git pull origin main` → Mengambil update terbaru dari GitHub.
- `git push origin --all` → Mengirim semua branch ke remote repository.
